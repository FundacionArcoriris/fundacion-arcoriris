# Fundación Ecológica Arcoiris

Sitio web estático listo para publicar gratis en GitHub Pages.

## Estructura
- `index.html`: contenido de la página.
- `styles.css`: diseño visual.
- `script.js`: menú móvil y año automático.
- `assets/`: logo, QR y fondo.

## Publicación
Sube todos estos archivos a un repositorio público en GitHub y activa GitHub Pages desde `Settings > Pages`.
